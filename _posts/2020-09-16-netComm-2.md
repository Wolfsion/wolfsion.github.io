# 计网其他先导

## 电路交换与分组交换

|                  |   电路交换   |     分组交换     |
| :--------------: | :----------: | :--------------: |
|      独占性      |     独占     |      非独占      |
|     传输效率     |      高      |        低        |
|      时序性      |   存在时序   |    不存在时序    |
|    端设备要求    |     简单     |       复杂       |
|  数据直达与可控  |    不可控    |       可控       |
|  链路建立与维护  |     需要     |      不需要      |
| 可靠性与冗余保证 |      差      |        好        |
|      利用率      |      低      |        高        |
|     传输信号     |   数字信号   |     数字信号     |
|                  | 需要终端匹配 | 具有存储转发时延 |
